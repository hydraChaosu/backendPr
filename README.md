## Installation

go to folder in which you will be having application and run

```bash
$ npm install
```

## Setup the app

export tables to db from shopSQLinit.sql

export data from data.sql

## Running the app

start xampp and run mysql service in it then run the app

```bash
# development
$ npm run start
```
